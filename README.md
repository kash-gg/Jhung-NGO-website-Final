# Zunj NGO Website 🌍

Official website for **Zunj Divyang Sanstha**, designed to establish a strong digital presence and share the organization’s initiatives online.  
Live at 👉 https://zunj-sanstha.vercel.app

---

## 🧱 Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** Supabase  
- **Deployment:** Vercel (Frontend), Render (Backend)

---

## 🚀 Features

- NGO overview and mission section  
- Campaign and donation pages  
- Responsive design for all screen sizes  
- Modular frontend and backend separation  
- Easy integration for future admin panel

---

## ⚙️ Setup Instructions

### Clone and Install

```bash
git clone https://github.com/kkbot122/Jhung-NGO-website-Final.git
cd Jhung-NGO-website-Final
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

App runs at http://localhost:5173.

### Backend
```bash
cd backend
npm install
npm run start
```

Server runs at http://localhost:5000.
